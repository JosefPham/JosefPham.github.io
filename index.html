2. Semesterproject

This project involves a case from the company EG Team Online to develop a public service system that digitally manages cases for disabled and vulnerable adults. In this project, we got familiar with JavaFX to design the frontend and to work with databases(PostgreSQL) to save persistent data.

4. Semesterproject

This project was about working with component-based software engineering. We utilized the NetBeans platform, and developed a system using NetBeans modules. The result was a game based on the classic game: Asteroids.


GitHub repository can be seen here.

4. Semesterproject, Independent Study Activity

This independent study activity was made in collaboration with the SDU Formula Student Team, SDU Vikings, who had a goal to make a remote-controlled car perform an acceleration test autonomously.


In this project, we developed and studied an approach for the detection and classification of multiple cones, that would be used in the acceleration test. The aim was to investigate the possibilites of using machine learning-based algorithms for cone detection for a driverless vehicle.


GitHub repository can be seen here.

5. Semesterproject

The purpose of this project was to explore and work wit the different web technologies taught in the semester: HTML, CSS, JavaScript, Node etc.


The project resulted in a webapplication build with angular. The frontend communicates with a server build with Node and Express, and a NoSQL to store and retrieve registered users.


GitHub repository can be seen here.

6. Semester, Bachelorproject

The bachelorproject was made in collaboration with the company Swarco Technology. The company develops and markets traffic controllers and communication equipment for traffic management systems and has customers in more than 50 countries.


We developed an Android application according to their desire and requirements - the application would help optimize the work flow in the company's production and also help to manage and track their storage.


The project ended with great success - we released a fully functioning application that is still being used in the company today.


GitHub repository can be seen here.
